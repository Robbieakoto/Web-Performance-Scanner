How We Did This

Methodology

🔬

Tool Used

Google PageSpeed Insights API v5, powered by Lighthouse 11. Industry standard for web performance measurement.

📱

Device Mode

All audits run in mobile mode — the most relevant simulation for Ghana where mobile internet dominates (~80% of users).

🌐

Network Simulation

Lighthouse simulates a mid-tier mobile connection (150 Kbps download, 675ms RTT) — representative of typical Ghanaian mobile conditions.

📅

Data Collection

Audits were run in April 2026. Results reflect one-time lab data and may vary with time of day and site updates.

🎯

Site Selection

20 high-traffic Ghanaian sites selected across 4 sectors: News, Banking/Finance, E-Commerce/Tech, and Government/Education.

📊

Scoring

Performance score is a weighted composite of FCP, LCP, Speed Index, TBT, and CLS — all measured by Lighthouse.
